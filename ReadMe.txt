1. Compile the solutions or make build and publish or deploy the sites (API and MVC application) to respective servers.

2. In the ContactAPI project, please change the appsettings.Development.json file to reflect the correct Database connection string for the key "ConStr" under AppSettings.

3. In the ContactUI project, please change the appsettings.Development.json file to reflect the correct API Service URL for the key "ServiceURL" under AppSettings.

4. Navigate to the project/folder ContactUtilities/DB - Run the script "ContactScript.sql" in the respective database for which connection string was set in Step 2.
